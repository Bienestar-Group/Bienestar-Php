JAVA WEB

JSP
Es un acrónimo de Java Server Pages, que en castellano vendría a decir algo como Páginas de Servidor Java.
DECLARACION
Las declaraciones de las variables o métodos se hacen de la forma: "<%! declaración; %>".
DIRECTIVAS
La directiva taglib indica al motor jsp que la página va a utilizar “tag libraries” o librerías de etiquetas.
SERVLETS
Los Servlets son programas escritos en Java y que dan una respuesta alternativa a la programación Web.
APPELTS
En Java, un applet es un programa que puede incrustarse en un documento HTML.
